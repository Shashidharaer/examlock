## Welcome to Your Statamic Entry

This is an example page showing how Statamic entries work with Inertia.js and React.

### Features

- **Markdown Content**: Write content in Markdown
- **Custom Fields**: Add any fields you need in the blueprint
- **Rich Data**: All entry data is passed to your React components

### Getting Started

1. Edit this entry in the Statamic Control Panel
2. Add more fields to the blueprint
3. Customize the entry.tsx component to display your content

You can access all entry data in your React component!
