      ## Enable Accessibility Features
      
      ExamLock is committed to providing an accessible platform for all users. Learn how to enable and configure accessibility features.
      
      ### Screen Reader Support
      - Compatible with JAWS, NVDA, and VoiceOver
      - Enable screen reader mode in Settings > Accessibility
      - Keyboard navigation fully supported
      
      ### Visual Accessibility
      - **High Contrast Mode**: Enhance visibility with high contrast themes
      - **Font Size**: Adjust text size from 100% to 200%
      - **Color Blind Mode**: Alternative color schemes for color vision deficiency
      - **Dark Mode**: Reduce eye strain with dark theme
      
      ### Keyboard Shortcuts
      - `Ctrl + /` - Open keyboard shortcuts help
      - `Tab` / `Shift + Tab` - Navigate between elements
      - `Enter` - Activate buttons and links
      - `Esc` - Close dialogs and modals
      
      ### Time Extensions
      - Configure extended time for assessments
      - Set custom time limits for individual students
      - Pause and resume functionality
      
      ### Alternative Input Methods
      - Voice input support
      - Switch access compatibility
      - Eye tracking device support
      
      ### How to Enable
      1. Go to **Settings** > **Accessibility**
      2. Toggle the features you need
      3. Customize settings for your preferences
      4. Save changes
      
      ### Support
      Contact our accessibility team at accessibility@examlock.com for assistance.
